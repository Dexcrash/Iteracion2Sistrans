Para ejecutar el proyecto siga los siguientes pasos:
-Importe la carpeta del proyecto en Eclipse. 
-Corra el proyecto en el servidor de WildFly 10x. 
-Cuando salga la pantalla que dice RotondAndes en Eclipse diríjase a Postman.
-Una vez en Postman presione Import y luego Choose Files. Escoja las Collections -que se encuentran en Docs/Collections postman. 
-Una vez allí puede correr cada una de las colecciones para probar los requerimientos funcionales. Si desea probar solo un ejemplo en específico, en el documento adjunto encuentra las URLs usadas para cada prueba, así como su JSON (si lo requiere) y lo que esta representa.


A continuación se presenta un ejemplo de ejecución:
1. Seleccione la colección RF1 - Registrar Usuario
2. Seleccione el item "Get de los usuarios en general"
3. De click en Send (El botón azul en la parte izquierda)
4. Observe los resultados.
5. De click en la pestaña Tests para ver si la prueba funciona correctamente.
